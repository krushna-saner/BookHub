# 📚 BookHub – Full Stack Bookstore Application

BookHub is a full-stack web application built using **Java (Spring Boot)** for backend and **React.js** for frontend.  
It is an online bookstore system with features like book browsing, search, cart management, and order placement.

---

## 🚀 Tech Stack

### Frontend:
- React.js
- JavaScript
- HTML5
- CSS3
- Bootstrap
- Axios
- React Router DOM

### Backend:
- Java
- Spring Boot
- Spring Data MongoDB
- REST APIs

### Database:
- MongoDB Atlas

---

## 📌 Features

- 🔍 Search books by name
- 📚 View books by categories
- 🛒 Add to cart functionality
- 📦 Order placement system
- 🔐 User authentication (if implemented)
- 📱 Fully responsive UI
- ⚡ REST API integration using Axios
